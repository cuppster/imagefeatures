imagefeatures
=============

A plug-in library for extracting features from images


Dependencies
------------

* numpy
* scipy
* scikit-image


Quickstart
----------

Clone this repository:

    $ git clone https://github.com/cuppster/imagefeatures
    
Run the tests

    $ cd imagefeatures
    $ python -m tests.features
    
Install

    $ python setup.py install
